# **App Name**: ShiftWise Payroll

## Core Features:

- Hybrid Shift Dashboard: Toggle between 12-hour and 9-hour presets for quick daily attendance logging and status tracking.
- Bulk Entry Tool: Mass-upload attendance for entire teams using a standardized CSV or structured UI grid to streamline high-volume entry.
- Precision Salary Calculator: Automatic calculation of time-based earnings based on the specific shift duration and base rates.
- AI Payroll Audit Tool: An intelligent tool that identifies shift patterns, potential payroll discrepancies, and cost forecasts based on historical logs.
- Daily Logs & Reporting: A database for recording specific clock-in/out timestamps and generating detailed employee statements.
- Employee Profile Hub: Manage individual employee data including default shift types (9 or 12 hours) and hourly pay rates.

## Style Guidelines:

- Primary color: Electric Cobalt (#6082F2), selected to evoke a sense of digital precision and technical reliability.
- Background: Deep Ink Slate (#0F1115), providing a high-contrast dark mode foundation that mimics premium dashboard environments.
- Accent color: Bright Cyan (#00E5FF), used for success indicators, active shift statuses, and highlighting financial calculations.
- Headline font: 'Space Grotesk', a geometric sans-serif that lends a machined, scientific aesthetic to the navigation. Body font: 'Inter', ensuring maximum readability for complex financial data and tables.
- Minimalist, stroke-based icons with subtle rounded edges to maintain a modern, professional tool-oriented visual language.
- A dense, modular grid layout focusing on table views and data density, allowing payroll managers to see high-level stats without scrolling.
- Swift, linear transitions for data updates to emphasize speed and immediate feedback when toggling shift types or updating rates.
